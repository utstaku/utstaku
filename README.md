# こんにちは

東京大学大学院で、プラズマ物理、数値シミュレーション、機械学習に関心を持って研究しています。物理モデリングとデータ駆動型手法を組み合わせ、複雑な力学系を理解・モデル化するための方法に興味があります。

趣味でKaggleをやっています!ソロで参加していることが多いですが、チームを組んでくれる方を常に募集しています!

## 研究関心

- プラズマ物理シミュレーション
- PDE の数値計算手法
- Vlasov-Poisson 方程式
- Landau damping / Two-stream instability
- Physics-informed machine learning
- Auto-Encoder / 潜在表現学習
- SINDy / シンボリック回帰 / 方程式発見
- 流体モデリング・closure 問題
- Neural operators / Fourier Neural Operator
- データ同化・状態空間モデル

研究では、機械学習を物理モデリングや科学計算にどのように活用できるかを探っています。特に、低次元モデリング、解釈可能な機械学習、物理システムの数理モデルを発見・改良するためのデータ駆動型手法に関心があります。

## 開発経験

研究や開発では主に Python を用いて、数値シミュレーション、機械学習モデルの実装、データ解析、可視化を行っています。

- 物理システムの数値シミュレーション・解析
- Auto-Encoder を用いた低次元表現学習
- PyTorch によるニューラルネットワークモデルの実装
- PySINDy / PySR を用いたデータ駆動型方程式発見
- Fourier Neural Operator などのニューラルオペレータの検討
- Docker / VS Code Remote SSH / Linux 環境での研究開発
- Git / GitHub を用いたコード管理
